# Demo Graphql 使用 JWT 實作登入的方式

### 指令
* cd .example.env .env 
* composer install
* php artisan key:gen
* php artisan migrate
* php db:seed

### Graphql-playground
```
/graphql-playground
```

### 實作關聯 
* user
* post
* commit
* jobs
